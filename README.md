# Vista Barber Company Website

Static GitHub Pages site for Vista Barber Company.

## Upload to GitHub Pages

1. Create or open your GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and the `assets` folder to the root of the repo.
3. Go to **Settings → Pages**.
4. Set the source to your main branch and root folder.
5. Save and wait for GitHub Pages to publish.

## Booking links
