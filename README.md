# Vista Barber Company Website

Static GitHub Pages site for Vista Barber Company.

## Upload to GitHub Pages

1. Create a GitHub repo.
2. Upload everything in this folder to the repo root.
3. Go to **Settings → Pages**.
4. Set source to the main branch / root folder.
5. Save and wait for GitHub Pages to publish.

## Notes

- The site uses local image files in `/assets`, including Vista Barber Company logo and Instagram/Wix-feed images.
- The generic Book buttons open a small location picker.
- Direct location buttons go to the Acuity booking links for Vista and San Marcos.
- Maps are embedded for both locations.
